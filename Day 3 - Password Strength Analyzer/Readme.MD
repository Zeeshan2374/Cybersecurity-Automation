# 🔐 Day 03 - Password Strength Analyzer

A simple Python-based tool that analyzes password strength using basic security rules and complexity checks.

---

## 📌 Features

- Password length analysis
- Uppercase and lowercase checks
- Number detection
- Special character detection
- Simple password strength rating

---

## 🛠️ Technologies Used

- Python
- Regular Expressions (Regex)
